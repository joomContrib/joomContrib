joomContrib
===========

 Extra packages for [Joomla Framework](http://framework.joomla.org/)

## App configuration

```YAML

debug: true

# Database
database:
    driver   : "mysqli"
    host     : "localhost"
    user     : "db_user"
    password : "db_user_password"
    database : "db_name"
    prefix   : "tables_prefix"


# System
system:
    session:
        storage: "storage type"
		storage_options: "storage options"

```
